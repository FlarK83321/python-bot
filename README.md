# python-bot

## Установка
	`git clone https://github.com/FlarK83321/python-bot.git`

## Директории
	В директории `account` и `group` нужно добавить файл `.access`
		или же с другим названием, но тогда *изменить название файла в файлах `__init__.py` этих директорий*

	В `.access` файле хранится ключ доступа (token)

### Получить токен
	`https://vkhost.github.io/`
